Welcome Employees

This employee portal is automatically deployed whenever developers push changes to GitHub.

CI/CD Pipeline

  • Developer pushes code to GitHub
  • GitHub Actions downloads latest source
  • HTML Validation runs automatically
  • GitHub connects to AWS EC2
  • Files are copied to Nginx Web Server
  • Website is refreshed automatically
